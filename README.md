# snehanshn.com

Personal portfolio site for Snehanshn Chowdhury.
Next.js App Router, TypeScript, Tailwind CSS v4, Framer Motion. Dark theme only.

## Develop

```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # production build
```

All copy and data live in `src/content.ts`; items marked `TODO(snehanshn)` are placeholders.
Design tokens and the hero canvas tuning are documented in `AGENTS.md`.
